## üöÄ COMPLETE INSTALLATION STEPS

### Step 1: Check Python Version
```bash
python --version
```
**Must be 3.7 or higher**

### Step 2: Install ALL Required Packages

**OPTION A: Install individually (if one fails, you'll know which)**
```bash
pip install numpy
pip install pandas
pip install scikit-learn
pip install plotly
pip install streamlit
pip install yfinance
```

**OPTION B: Install all at once (Recommended)**
```bash
pip install numpy pandas scikit-learn plotly streamlit yfinance
```

**OPTION C: Using requirements file**
```bash
pip install -r requirements_streamlit.txt
```

### Step 3: Verify Installation
```bash
pip list | grep -E "streamlit|scikit-learn|plotly|numpy|pandas"
```

Should show all packages listed.

### Step 4: Run the App
```bash
streamlit run streamlit_app.py
```

**Expected output:**
```
Collecting usage statistics...
Local URL: http://localhost:8501
Network URL: http://192.168.x.x:8501

  You can now view your Streamlit app in your browser.
```

Browser opens automatically! ‚úÖ

## ‚ö†Ô∏è COMMON ERRORS & FIXES

### Error 1: "pip: command not found"

**Solution:** Use `pip3` instead (especially on Mac/Linux):
```bash
pip3 install streamlit numpy pandas scikit-learn plotly yfinance
```

### Error 2: "No module named 'sklearn'"

**Solution:** scikit-learn not installed:
```bash
pip install scikit-learn
```

### Error 3: "Port 8501 already in use"

**Solution:** Use a different port:
```bash
streamlit run streamlit_app.py --server.port 8502
```

### Error 4: "ModuleNotFoundError" for any package

**Solution:** Install that specific package:
```bash
# Example for plotly
pip install plotly

# Example for numpy
pip install numpy

# Example for pandas
pip install pandas
```

### Error 5: "CERTIFICATE_VERIFY_FAILED" during installation

**Solution:** Bypass certificate check:
```bash
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org streamlit numpy pandas scikit-learn plotly yfinance
```

### Error 6: "PermissionError" or "Access Denied"

**Solution:** Install for user only:
```bash
pip install --user streamlit numpy pandas scikit-learn plotly yfinance
```

Or use administrator/sudo (Mac/Linux):
```bash
sudo pip3 install streamlit numpy pandas scikit-learn plotly yfinance
```

## üìã WHAT TO DO IF STILL ERRORS

### Step 1: Check pip is updated
```bash
pip install --upgrade pip
```

### Step 2: Try fresh installation
```bash
pip install --no-cache-dir streamlit numpy pandas scikit-learn plotly yfinance
```

### Step 3: Uninstall and reinstall
```bash
pip uninstall streamlit numpy pandas scikit-learn plotly yfinance
pip install streamlit numpy pandas scikit-learn plotly yfinance
```

### Step 4: Check Python environment
```bash
which python           # Mac/Linux
where python          # Windows
```

Make sure it's the right Python (not in virtual environment accidentally)
