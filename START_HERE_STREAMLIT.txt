ğŸ”§ TROUBLESHOOTING

PROBLEM: App is slow
SOLUTION: Use synthetic data (faster) or reduce lookback period

PROBLEM: Charts not showing
SOLUTION: Try different browser (Chrome/Firefox)
         Or run: streamlit run streamlit_app.py --logger.level=debug

PROBLEM: Downloaded data empty
SOLUTION: Check ticker spelling (e.g., AAPL not Apple)
         Use major stocks: AAPL, MSFT, GOOGL, TSLA, AMZN

PROBLEM: Need to restart app
SOLUTION: Ctrl+C in terminal to stop
         streamlit run streamlit_app.py to start again
